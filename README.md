# Zainab Imtiaz - Developer Portfolio

A responsive personal portfolio website developed as part of a web development assignment. The website showcases my skills, projects, and contact information using modern frontend technologies.

https://assignment-2-two-eta.vercel.app/

## Project Overview

This portfolio website includes:

- Responsive navigation bar
- Hero section
- About section
- Skills section
- Projects section
- Contact form with JavaScript validation
- Smooth scrolling navigation
- Responsive design for desktop, tablet, and mobile devices
